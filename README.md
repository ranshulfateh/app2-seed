# Introduction


In order to start the seed use:


```bash
git clone --depth 1 https://github.com/ranshulfateh/app2-seed.git

# install the project's dependencies
npm install
# watches your files and uses livereload by default
npm start
# api document for the app
# npm run build.docs

# dev build
npm run build.dev
# prod build
npm run build.prod
```
